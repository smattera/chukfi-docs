## Why Chukfi?

Chukfi is a headless CMS built from the ground up for **speed, simplicity, and AI-native workflows**. Three non-negotiable pillars:

<p>Docker PostgreSQL locally, AWS RDS in production. Same engine, same migrations, same SQL. No SQLite surprises at deploy time.</p>

## AI-Native by Design

Every CLI command outputs structured JSON. This means **AI agents can operate Chukfi as easily as humans** — no screen scraping, no brittle parsing.

## Get Started in 30 Seconds

```bash
npm install @chukfi/cli
npx chukfi dev
```

That's it. Chukfi detects Docker, spawns PostgreSQL, runs migrations, and starts the API on port 8080. Your CMS is live.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| **Backend** | Rust (native binary via Axum) |
| **Frontend** | Astro 7 + React islands |
| **Database** | PostgreSQL (Docker local / AWS RDS) |
| **Auth** | Magic Link + Microsoft Entra ID |
| **AI** | Amazon Bedrock (Claude 3.5 Haiku) |
| **Media** | S3 / Local filesystem |
| **Deploy** | AWS CDK (ECS Fargate + RDS + S3 + CloudFront) |

## Frequently Asked Questions

<div class="faq-accordion">

<details class="faq-item">
<summary>Do I need to know Rust to use Chukfi?</summary>
<div class="faq-answer">
No. Chukfi is distributed as a pre-compiled native binary wrapped in npm packages. You interact with it entirely via CLI commands — no Rust toolchain, no compilation, no Cargo.
</div>
</details>

<details class="faq-item">
<summary>How does the one-command deploy work?</summary>
<div class="faq-answer">
<code>npx chukfi deploy</code> uses AWS CDK under the hood to provision ECS Fargate, RDS PostgreSQL, S3, CloudFront, and SES. It reads your local config and maps it to production — same engine, same migrations, same behavior.
</div>
</details>

<details class="faq-item">
<summary>Can I use Chukfi without AWS?</summary>
<div class="faq-answer">
The core API runs anywhere Docker or a Linux binary works. The automated one-command deploy is optimized for AWS, but you can self-host the binary on any platform that runs PostgreSQL.
</div>
</details>

<details class="faq-item">
<summary>What makes it "AI-native"?</summary>
<div class="faq-answer">
Every CLI command accepts <code>--json</code> for structured output designed for LLM tool-calling. The Teams bot lets staff manage content via chat. All AI-driven changes are forced to draft — humans always approve before publishing.
</div>
</details>

<details class="faq-item">
<summary>Is there a free tier?</summary>
<div class="faq-answer">
Yes. The entire AWS stack fits within Free Tier limits for hobby projects. Chukfi itself is MIT-licensed and the CLI is free via npm.
</div>
</details>
