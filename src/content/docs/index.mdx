## Why Chukfi?

Chukfi is a headless CMS purpose-built for solo developers shipping Astro sites. Three non-negotiable pillars:

- **CLI-First Distribution** — The Rust binary ships as platform-specific npm packages. `npx chukfi` — no Rust toolchain, no build steps, no operating knowledge required.
- **Local-to-Prod Parity** — Local dev uses Docker-managed PostgreSQL; production uses AWS RDS PostgreSQL. Same engine, same migrations, same SQL. No SQLite surprises.
- **One-Command Deploy** — `npx chukfi deploy` provisions the full AWS stack via CDK. One domain, one bill, no hybrid hosting. Free Tier eligible.

## Get Started in 30 Seconds

```bash
npm install @chukfi/cli
npx chukfi dev
```

That's it. Chukfi detects Docker, spawns PostgreSQL, runs migrations, and starts the API on port 8080.

## Built for AI Agents

Every CLI command supports `--json` output for machine-parseable responses. Chukfi is designed to be operated by AI agents as easily as by humans:

```bash
# Create a draft blog post
npx chukfi content create \
  --type blog-posts \
  --title "Hello World" \
  --fields '{"body":"<p>First post!</p>"}' \
  --status draft \
  --json

# Upload media
npx chukfi media upload \
  --path ./images/hero.png \
  --alt "Hero banner" \
  --json

# Deploy to production
npx chukfi site deploy \
  --dir ./frontend \
  --project my-site \
  --yes \
  --json
```

## Tech Stack

| Layer | Technology |
|-------|-----------|
| **Backend** | Rust (native binary via Axum) |
| **Frontend** | Astro 7 + React islands |
| **Database** | PostgreSQL (Docker local / AWS RDS) |
| **Auth** | Magic Link + Microsoft Entra ID |
| **Media** | S3 / Local filesystem |
| **Deploy** | AWS CDK / Cloudflare Pages |
